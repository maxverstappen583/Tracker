# Max Portfolio Website

This repository contains the source code for my personal portfolio website, including
Discord presence, activity status, and Spotify integration.

## ⚠️ Project Status: Discontinued

**This project is discontinued and is no longer actively maintained.**

- No new features will be added
- Bugs may not be fixed
- APIs, presence features, or integrations may break over time
- The code is provided **as-is** for reference or learning purposes

## Purpose

This project was originally built to:
- Display my developer portfolio
- Show live Discord presence (status, activity, Spotify)
- Experiment with UI animations and real-time data

## Tech Stack

- HTML
- CSS
- JavaScript
- Vercel Serverless Functions
- Lanyard API (Discord presence)

## Notes

If you fork or reuse this project:
- You may need to update API endpoints
- Discord presence behavior depends on third-party services
- Deployment issues on Vercel are possible due to API limitations
